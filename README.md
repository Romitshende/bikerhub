# BikerHub 🏍️

A high-performance digital atelier for motorcycle enthusiasts. BikerHub bridges the gap between raw track telemetry and community ride coordination, designed for riders who value precision and style.

## Key Features
* **Elite Garage:** Showcase your machine's modifications and specifications.
* **Ride Coordination:** Join upcoming community rides and track attendee rosters.
* **Performance Focus:** Clean, minimalist UI built for high-performance aesthetic.

## Tech Stack
* **Framework:** Next.js (App Router)
* **Styling:** Tailwind CSS
* **Deployment:** Vercel

## Getting Started
1. Clone this repository:
   ```bash
   git clone [https://github.com/Romitshende/bikerhub.git](https://github.com/Romitshende/bikerhub.git)
   ```
2. Install dependencies:
  ```bash
   npm install
  ```
3. Run the development server:
  ```bash
   npm run dev
  ```
4. Open 
  ```bash
   http://localhost:3000
   ```
   in your browser.
